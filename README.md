# AI Travel Assistant – Backend

This backend is powered by **Phi** agents running on **Groq’s LLM (LLaMA 3.3 70B)**. It processes travel-related prompts and financial queries by integrating tools like `YFinance`, `DuckDuckGo`, and custom functions.

---

## ✨ Features

- 🧠 AI agent handles natural language travel queries
- 🗺️ Suggests itineraries, places to visit, and travel plans
- 📉 Optional financial tool integration with YFinance
- 🔁 Custom functions for symbol resolution and formatting
- 🌍 Web and finance agents working as a team

---

## 🧱 Tech Stack

- Python
- Groq LLaMA 3.3-70B
- dotenv for env management
